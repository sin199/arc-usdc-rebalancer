# Arc Builder Notes

This repository is a public Arc Testnet treasury demo. It is meant to read like an active Arc builder project, not a generic dashboard.

## Builder milestones

- Connected the dashboard to Arc Testnet policy reads and owner-gated writes.
- Added a public demo mode so visitors can inspect the flow without a wallet.
- Integrated an Arc agent identity, validation state, and operational brief.
- Added Circle developer-controlled wallet and Gateway readiness surfaces.
- Split the dashboard into public demo mode and live operator mode.

## What reviewers should see first

1. The live dashboard at `https://web-eight-chi-99.vercel.app/dashboard`.
2. The Arc agent panel and brief panel.
3. The Circle line and wallet readiness state.
4. The treasury policy and executor controls.
5. The public demo mode that stays usable without wallet access.

## Positioning

- Arc-native treasury workflow.
- USDC-first.
- Agent-assisted.
- Public demo for visitors.
- Signed operator actions for live execution.
